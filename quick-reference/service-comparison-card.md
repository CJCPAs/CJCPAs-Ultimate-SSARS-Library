## When to Use Each Service

### Preparation (AR-C 70)
- Internal use only
- No third-party reliance needed
- Most cost-effective option
- You provide bookkeeping services

### Compilation (AR-C 80)
- Third parties may see statements
- No assurance needed
- Bank doesn't require review
- Need formal CPA report

### Review (AR-C 90)
- Bank/creditor requires assurance
- Third parties need credibility
- Higher level of confidence needed
- Independence can be maintained

## Independence Decision Tree

```
START: What service is requested?
│
├── PREPARATION → Independence NOT required
│                  No disclosure needed
│
├── COMPILATION → Independence NOT required
│   │              BUT if not independent:
│   │              MUST disclose in report
│   │              CANNOT disclose reason
│   │
│   └── Are you independent?
│       ├── YES → Standard report
│       └── NO  → Add: "I am (We are) not independent
│                       with respect to [Company]."
│
└── REVIEW → Independence IS REQUIRED
    │
    └── Are you independent?
        ├── YES → Proceed with review
        └── NO  → CANNOT perform review
                  Consider stepping down to compilation
```

## Key Requirements Summary

### All SSARS Engagements
- Engagement letter (signed)
- Documentation
- Professional judgment
- Ethical requirements

### Preparation Only
- Legend on EACH page: "No assurance is provided"

### Compilation Only
- Report required
- Read financial statements
- Understand the entity
- Independence disclosure if not independent

### Review Only
- Independence (mandatory)
- Analytical procedures
- Inquiries of management
- Management representation letter
- Limited assurance conclusion

---

## Report Language Quick Reference

### Compilation - Key Phrase
> "I (We) do not express an opinion, a conclusion, nor provide any form of assurance on these financial statements."

### Review - Key Phrase
> "Based on my (our) review, I am (we are) not aware of any material modifications that should be made to the accompanying financial statements..."

## Common Mistakes to Avoid

| Service | Common Mistake | Correction |
|---------|---------------|------------|
| Preparation | Missing legend | Add to EACH page |
| Compilation | No independence disclosure | Add when not independent |
| Compilation | Disclosing reason for lack of independence | Remove reason - just state "not independent" |
| Review | Performing when not independent | Cannot issue - step down to compilation |
| Review | Missing rep letter | Required - cannot issue without it |
| All | No engagement letter | Always get signed letter first |
