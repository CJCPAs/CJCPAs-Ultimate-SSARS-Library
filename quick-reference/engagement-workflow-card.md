## Service-Specific Workflow Details

### Preparation Engagement (AR-C 70)

```
1. ACCEPT    → Evaluate client, get signed engagement letter
2. UNDERSTAND → Know the framework and policies
3. PREPARE   → Format financial statements from client data
4. LEGEND    → Add "No assurance provided" to EACH page
5. DELIVER   → Give statements to client (no report)
6. DOCUMENT  → Assemble file within 60 days
```

### Compilation Engagement (AR-C 80)

```
1. ACCEPT    → Evaluate client, assess independence, get signed letter
2. UNDERSTAND → Know entity, industry, framework, policies
3. COMPILE   → Apply expertise to present statements
4. READ      → Read for obvious errors, consider form
5. REPORT    → Issue compilation report (modify if needed)
6. DOCUMENT  → Assemble file within 60 days
```

### Review Engagement (AR-C 90)

```
1. ACCEPT    → Evaluate client, CONFIRM INDEPENDENCE, get signed letter
2. UNDERSTAND → Know entity, environment, internal control, risks
3. ANALYZE   → Perform analytical procedures on all material areas
4. INQUIRE   → Make required inquiries of management
5. INVESTIGATE → Follow up on differences and responses
6. REPRESENT → Obtain signed management representation letter
7. REPORT    → Issue review report with conclusion
8. DOCUMENT  → Assemble file within 60 days
```

## Critical Checkpoints

### Before Starting Work
- [ ] Engagement letter signed?
- [ ] Independence assessed? (Reviews: confirmed?)
- [ ] Competence to perform?

### Before Issuing Report/Statements
- [ ] All procedures completed?
- [ ] Documentation sufficient?
- [ ] Report/legend appropriate for circumstances?
- [ ] (Review) Rep letter obtained and dated correctly?

### After Completion
- [ ] File assembled within 60 days?
- [ ] All documentation retained?
- [ ] Invoice sent?

## Documentation Assembly Timeline

```
Report Date ───────────────────────────────────────────► 60 Days
     │                                                      │
     │  ◄────── Assemble documentation during this period ──────►
     │                                                      │
     ▼                                                      ▼
  Issue                                              Assembly
  Report                                             Deadline
```

**After assembly deadline:**
- No new documentation can be added
- Only administrative changes permitted
- Retain for minimum 5 years
