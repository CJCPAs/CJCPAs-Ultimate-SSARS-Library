# Peer Review Alerts and Common Deficiencies

## Overview

This guide compiles common peer review deficiencies found in SSARS engagements. Understanding these deficiencies helps practitioners avoid them and improve engagement quality.

---

## Most Common Deficiencies by Engagement Type

### Preparation Engagements (AR-C 70)

| Rank | Deficiency | Description | How to Avoid |
|------|------------|-------------|--------------|
| 1 | **Missing Legend** | Legend not on each page of financial statements | Add legend to header/footer of each page |
| 2 | **Incomplete Legend** | Legend doesn't state "no assurance provided" | Use exact required language |
| 3 | **No Engagement Letter** | Engagement letter not obtained | Get signed letter before starting |
| 4 | **Missing Required Elements** | Engagement letter missing required elements | Use checklist for letter elements |
| 5 | **Insufficient Documentation** | File lacks evidence of procedures | Document understanding and work performed |

### Compilation Engagements (AR-C 80)

| Rank | Deficiency | Description | How to Avoid |
|------|------------|-------------|--------------|
| 1 | **No Engagement Letter** | Engagement letter not obtained or not signed | Require signed letter before starting |
| 2 | **Independence Not Documented** | No evidence of independence assessment | Document assessment in every engagement |
| 3 | **Independence Not Disclosed** | Not independent but report doesn't disclose | Add disclosure when not independent |
| 4 | **Old Report Language** | Using outdated report format | Use current SSARS language |
| 5 | **No Evidence of Reading** | No documentation of reading financial statements | Document review for obvious errors |
| 6 | **Insufficient Understanding** | Entity understanding not documented | Document industry, operations, policies |
| 7 | **Disclosure Omission Language** | Incorrect or incomplete omission language | Use exact required wording |
| 8 | **Departure Not Disclosed** | Known departure not disclosed in report | Modify report for all known departures |

### Review Engagements (AR-C 90)

| Rank | Deficiency | Description | How to Avoid |
|------|------------|-------------|--------------|
| 1 | **Insufficient Analytical Procedures** | Not performed for all material areas | Perform procedures on all significant areas |
| 2 | **No Expectations Documented** | Expectations not developed/documented | Document expectations before comparing |
| 3 | **Missing Management Rep Letter** | Required letter not obtained | Cannot issue report without letter |
| 4 | **Rep Letter Dating Error** | Letter not dated same as report | Match dates exactly |
| 5 | **Inadequate Inquiries** | Required inquiries not made or documented | Use checklist for required inquiries |
| 6 | **Independence Not Documented** | No documentation of independence | Document assessment - required for reviews |
| 7 | **Insufficient Follow-up** | Differences not investigated | Document investigation of all differences |
| 8 | **Missing Report Elements** | Required report sections missing | Use complete report template |

## Detailed Analysis of Key Deficiencies

### 1. Engagement Letter Deficiencies

#### Common Issues

- No engagement letter on file
- Letter not signed by client
- Letter dated after report date
- Missing required elements

#### Required Elements Comparison

| Element | Preparation | Compilation | Review |
|---------|-------------|-------------|--------|
| Objective and scope | Required | Required | Required |
| Management responsibilities | Required | Required | Required |
| Accountant responsibilities | Required | Required | Required |
| Limitations | Required | Required | Required |
| Financial framework | Required | Required | Required |
| Legend requirement | Required | N/A | N/A |
| Report description | N/A | Required | Required |
| Management rep letter | N/A | N/A | Required |

#### Prevention

- [ ] Use engagement letter templates with all required elements
- [ ] Obtain signed letter before starting work
- [ ] Document if client refuses to sign
- [ ] Review letter for completeness before filing

---

### 2. Independence Deficiencies

#### Compilation Engagements

**Issue:** Failure to document independence assessment or disclose lack of independence

**Requirements:**
- Document independence evaluation for every engagement
- If not independent, disclose in report: "I am (We are) not independent with respect to [Company Name]."
- Do NOT disclose reason for lack of independence

**Prevention:**
- [ ] Complete independence checklist each engagement
- [ ] Document threats identified and safeguards applied
- [ ] Modify report when not independent

#### Review Engagements

**Issue:** Performing review when not independent or not documenting independence

**Requirements:**
- Independence is REQUIRED - cannot issue review if not independent
- Must document assessment and conclusion

**Prevention:**
- [ ] Evaluate independence before accepting engagement
- [ ] If independence impaired, step down to compilation
- [ ] Document assessment thoroughly

---

### 3. Analytical Procedures Deficiencies (Review)

#### Common Issues

- Procedures not performed on all material accounts
- No expectations developed
- Expectations not documented
- Differences not investigated
- Conclusions not documented

#### Requirements

Per AR-C 90.25: "The accountant should apply analytical procedures to the financial statements."

Documentation must include:
- Expectations and basis for expectations
- Results of comparisons
- Investigation of significant differences
- Conclusions reached

#### Prevention

- [ ] Use analytical procedures checklist
- [ ] Document expectations BEFORE performing procedures
- [ ] Calculate expected amounts
- [ ] Document actual vs. expected
- [ ] Investigate ALL significant differences
- [ ] Document management explanations
- [ ] Document conclusions

---

### 4. Management Representation Letter Deficiencies (Review)

#### Common Issues

| Issue | Frequency | Impact |
|-------|-----------|--------|
| Missing letter | High | Cannot issue report |
| Wrong date | High | Material deficiency |
| Missing representations | Medium | May not cover all required areas |
| Unsigned | Medium | Letter not valid |
| Wrong signer | Low | May not have authority |

#### Requirements

- Letter is REQUIRED for all review engagements
- Must be dated same date as review report
- Must include all required representations
- Must be signed by appropriate management

#### Prevention

- [ ] Use standard rep letter template
- [ ] Date letter same as report date
- [ ] Verify all required representations included
- [ ] Obtain signature before issuing report
- [ ] Verify signer has appropriate authority

---

### 5. Report Language Deficiencies

#### Compilation Report Issues

| Issue | Description |
|-------|-------------|
| Old format | Using pre-SSARS 21 language |
| Missing "SSARS" reference | Doesn't reference current standards |
| Wrong independence language | Incorrect disclosure wording |
| Missing omission language | Disclosure omission not properly described |

#### Review Report Issues

| Issue | Description |
|-------|-------------|
| Missing sections | Report doesn't have required headed sections |
| No independence statement | Required statement omitted |
| Wrong conclusion language | Not using proper limited assurance language |
| Emphasis paragraph errors | "Not modified" statement missing |

#### Prevention

- [ ] Use current report templates
- [ ] Compare report to standard before issuing
- [ ] Have second person review report
- [ ] Update templates when standards change

## Self-Assessment Checklist

### Before Issuing Any SSARS Report

#### Preparation Engagements
- [ ] Engagement letter signed and on file
- [ ] Legend on EACH page of financial statements
- [ ] Legend contains "no assurance provided" language
- [ ] Documentation supports engagement

#### Compilation Engagements
- [ ] Engagement letter signed and on file
- [ ] Independence assessed and documented
- [ ] If not independent, disclosed in report
- [ ] Evidence of reading financial statements
- [ ] Understanding of entity documented
- [ ] Report uses current SSARS language
- [ ] All known departures disclosed

#### Review Engagements
- [ ] Engagement letter signed and on file
- [ ] Independence confirmed and documented
- [ ] Analytical procedures performed on all material areas
- [ ] Expectations developed and documented
- [ ] Differences investigated and documented
- [ ] All required inquiries made and documented
- [ ] Management representation letter obtained
- [ ] Rep letter dated same as report
- [ ] Report includes all required sections
- [ ] Independence statement in report
- [ ] Conclusion uses proper limited assurance language

## Quality Control Recommendations

### Firm-Level Actions

1. **Annual training** on SSARS requirements
2. **Template updates** when standards change
3. **Checklist usage** mandatory for all engagements
4. **Second review** before report issuance
5. **Self-inspection** of recent engagements
6. **Monitor peer review alerts** for industry trends

### Engagement-Level Actions

1. **Use checklists** - Don't rely on memory
2. **Complete as you go** - Not at the end
3. **Document thoroughly** - Another accountant should understand
4. **Review before issuing** - Fresh eyes catch errors
5. **Keep templates current** - Update with new standards
