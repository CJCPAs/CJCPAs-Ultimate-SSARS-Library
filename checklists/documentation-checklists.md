# Documentation Checklists for SSARS Engagements

## Overview

Proper documentation is essential for all SSARS engagements. This comprehensive checklist covers documentation requirements for preparation, compilation, and review engagements.

---

## General Documentation Requirements (AR-C 60)

### Standard for All SSARS Engagements

Documentation should be sufficient to enable an **experienced accountant**, having no previous connection with the engagement, to understand:

- [ ] The nature, timing, and extent of procedures performed
- [ ] The results of procedures performed
- [ ] Significant findings or issues arising
- [ ] Conclusions reached
- [ ] Significant professional judgments made

### Documentation Timing

- [ ] Documentation assembled within **60 days** of report date
- [ ] No documentation added after assembly deadline (except administrative changes)
- [ ] Changes after assembly appropriately documented

### Retention Requirements

- [ ] Minimum **5 years** from report date (check state board requirements)
- [ ] Consider firm policy requirements
- [ ] Consider insurance requirements
- [ ] Consider client agreements

## Preparation Engagement Documentation (AR-C 70)

### Required Documentation

#### Engagement Agreement
- [ ] Engagement letter signed by client
- [ ] Engagement letter includes:
  - [ ] Objective and scope of engagement
  - [ ] Management's responsibilities
  - [ ] Accountant's responsibilities
  - [ ] Limitations of engagement
  - [ ] Financial reporting framework
  - [ ] Legend requirement

#### Financial Statements
- [ ] Copy of prepared financial statements
- [ ] Legend on each page: "No assurance is provided on these financial statements"
- [ ] Financial reporting framework correctly identified

#### Understanding of Framework
- [ ] Documentation of accountant's understanding of:
  - [ ] Applicable financial reporting framework
  - [ ] Significant accounting policies
  - [ ] Nature of entity's business activities

#### Significant Matters
- [ ] Any significant matters arising during engagement
- [ ] Communications with management
- [ ] Issues resolved and how resolved

## Compilation Engagement Documentation (AR-C 80)

### Required Documentation

#### Engagement Agreement
- [ ] Engagement letter signed by client
- [ ] Engagement letter includes:
  - [ ] Objective and scope of engagement
  - [ ] Management's responsibilities
  - [ ] Accountant's responsibilities
  - [ ] Limitations of engagement
  - [ ] Financial reporting framework
  - [ ] Expected form and content of report

#### Understanding of Entity
- [ ] Nature of entity and its business
- [ ] Industry considerations
- [ ] Organizational structure
- [ ] Significant accounting policies
- [ ] Sufficient to identify where misstatements may arise

#### Reading the Financial Statements
- [ ] Evidence that accountant read the financial statements
- [ ] Consideration of whether statements appear appropriate in form
- [ ] Consideration of whether statements are free from obvious material errors

#### Financial Statements and Report
- [ ] Copy of compiled financial statements
- [ ] Copy of compilation report issued
- [ ] Appropriate report language for circumstances

#### Independence Assessment
- [ ] Documentation of independence evaluation
- [ ] If not independent:
  - [ ] Documentation of lack of independence
  - [ ] Report modified to disclose

#### Significant Matters
- [ ] Any departures from framework identified
- [ ] Communications with management about departures
- [ ] Resolution of matters
- [ ] Report modifications for departures

#### Optional but Recommended
- [ ] Management representation letter (not required but recommended)
- [ ] Summary of significant accounting policies
- [ ] Working trial balance
- [ ] Adjusting entries

## Review Engagement Documentation (AR-C 90)

### Required Documentation

#### Engagement Agreement
- [ ] Engagement letter signed by client
- [ ] Engagement letter includes:
  - [ ] Objective and scope of engagement
  - [ ] Management's responsibilities
  - [ ] Accountant's responsibilities
  - [ ] Limitations of engagement
  - [ ] Financial reporting framework
  - [ ] Expected form of report
  - [ ] Management representation letter requirement

#### Independence
- [ ] Documentation that accountant is independent
- [ ] Independence assessed and confirmed
- [ ] No threats identified OR safeguards applied

#### Understanding of Entity and Environment
- [ ] Industry factors
- [ ] Entity operations and activities
- [ ] Ownership and governance structure
- [ ] Accounting policies and practices
- [ ] Understanding of internal control (general)
- [ ] Areas of higher risk identified

#### Analytical Procedures
- [ ] Procedures designed to identify unusual relationships
- [ ] For each material account/area:
  - [ ] Expectations developed
  - [ ] Basis for expectations documented
  - [ ] Procedures performed
  - [ ] Results obtained
  - [ ] Comparison to expectations
  - [ ] Investigation of differences
  - [ ] Conclusions reached

#### Inquiries
- [ ] Inquiries made of management documented
- [ ] Required inquiry topics covered:
  - [ ] Financial statement preparation
  - [ ] Changes in business activities
  - [ ] Subsequent events
  - [ ] Fraud or suspected fraud
  - [ ] Noncompliance with laws
  - [ ] Related party transactions
  - [ ] Litigation, claims, assessments
  - [ ] Going concern considerations
- [ ] Responses to inquiries documented
- [ ] Follow-up on responses

#### Management Representation Letter
- [ ] Signed representation letter obtained
- [ ] Dated same date as review report
- [ ] Includes all required representations
- [ ] Signed by appropriate management

#### Financial Statements and Report
- [ ] Copy of reviewed financial statements
- [ ] Complete notes to financial statements
- [ ] Review report
- [ ] Appropriate conclusion (unmodified, qualified, adverse)

#### Significant Matters
- [ ] All significant matters documented
- [ ] Communications with management documented
- [ ] Departures identified and addressed
- [ ] Emphasis of matter considerations

## Common Documentation Deficiencies

### Preparation Engagements
1. No engagement letter
2. Missing legend on financial statement pages
3. Insufficient documentation of understanding

### Compilation Engagements
1. No engagement letter
2. No evidence of reading financial statements
3. Missing independence assessment
4. Independence disclosure omitted when not independent
5. Inadequate understanding of entity documented

### Review Engagements
1. No engagement letter
2. Independence not documented
3. Insufficient analytical procedures
4. No expectations documented for analytical procedures
5. Missing management representation letter
6. Representation letter dated incorrectly
7. Inquiries not documented
8. No follow-up on differences documented

## Documentation Quality Review Checklist

### Before Issuing Report

- [ ] All required documentation complete
- [ ] Documentation supports conclusions
- [ ] Dates are appropriate
- [ ] Signatures obtained where required
- [ ] Report language matches circumstances
- [ ] Cross-references complete
- [ ] No loose ends or unanswered questions

### Final Assembly

- [ ] File organized logically
- [ ] Index/table of contents prepared
- [ ] All documents properly dated
- [ ] Preparer/reviewer sign-offs complete
- [ ] Electronic files properly named/stored
- [ ] Backup copies created
